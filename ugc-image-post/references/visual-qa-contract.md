# 图文视觉 QA 合同

逐页检查：

- 页面数量、顺序和 3:4 尺寸
- 产品包装、颜色、Logo、容量和标签
- 人物与交互场景的自然度
- 中文是否正确、清晰、无溢出和遮挡
- 标题层级、留白和整组视觉一致性
- 是否出现平台 UI、水印、对标品牌资产或额外 Logo
- 页面是否表达内容方案指定的功能

`minor` 不阻止通过；存在 `major` 或 `critical` 时 `pass` 必须为 `false`，
并为每个重大问题提供只纠正该问题的 `correction_prompt`。

自动纠错限制：

- 整组最多两页
- 同一页最多一次
- 纠错结果必须重新审核
- 超过限制时停止，不把失败页面标记为完成
