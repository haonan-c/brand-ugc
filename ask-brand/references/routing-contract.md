# Ask Brand 路由合同

## 明确意图

- 图文、小红书、图片、笔记、封面、内页、Carousel → `ugc-image-post`
- 视频、短视频、分镜、Seedance、Storyboard → `ugc-storyboard`
- 品牌档案、品牌规范、品牌语气、Brand Profile → `brand-profile`

明确意图不重复询问路径，只检查必填素材。

## 模糊意图

- 只有完整图文素材 → 推荐 `ugc-image-post`
- 只有完整视频素材 → 推荐 `ugc-storyboard`
- 两组素材都完整 → 推荐先用图文验证表达，但必须确认
- 素材不足且意图不明 → 询问先做图文还是短视频

一次回复只能提出一个路由问题。用户没有明确要求时，不同时执行两个生产 Skill。

## 必填素材

`ugc-image-post`：

- 1–9 张有顺序的对标图片
- 对标文案
- 产品图

`ugc-storyboard`：

- 对标视频
- 产品图

品牌档案对两个生产路径都是可选输入。
